1) JSONPlaceholder – Fake Users, Posts, Comments

Perfect for CRUD and async/await practice.

Endpoint examples:

Users → https://jsonplaceholder.typicode.com/users

Posts → https://jsonplaceholder.typicode.com/posts

Comments → https://jsonplaceholder.typicode.com/comments

2) DummyJSON – Products, Carts, Auth Simulation

Real-looking ecommerce data.

Endpoints:

Products → https://dummyjson.com/products

Users → https://dummyjson.com/users

Cart → https://dummyjson.com/carts

3) Cat Fact API – Random Facts

Fun API to teach random fetch.

Endpoint:

https://catfact.ninja/fact

4) Dog API – Dog Images

Great for displaying images dynamically.

Endpoint:

Random dog → https://dog.ceo/api/breeds/image/random

5) PokeAPI – Pokémon Data

Perfect for looping, fetching detail pages, nested objects.

Endpoint:

https://pokeapi.co/api/v2/pokemon

6) Advice Slip API – Random Motivational Lines

Very simple GET request practice.

Endpoint:

    https://api.adviceslip.com/advice

7) Public Holiday API – Countries Holiday Info

Good for date filtering & mapping.

Example:

https://date.nager.at/api/v3/PublicHolidays/2024/US

8) Open-Meteo – Free Weather API (No Key Needed)

Real-time weather data.
(Using latitude & longitude)

Example:

https://api.open-meteo.com/v1/forecast?latitude=24.8607&longitude=67.0011&current_weather=true

9) Bored API – Random Activity Suggestions

Simple & fun.

Endpoint:

https://www.boredapi.com/api/activity

10) SpaceX API – Rockets, Launches, Missions

Great for real JSON objects with depth.

Example:

https://api.spacexdata.com/v4/launches/latest
